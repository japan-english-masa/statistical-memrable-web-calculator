Statistical Functions Description

yi=β0+β1xi

You can get (β0,β1) value for least square method.

yi=β0+xi

You can get β0 value for least square method only using β0.

yi=β1xi

You can get β1 value for least square method only using β1.

x-

You can get the Mean for x values.

Sxx

You can get the Variance for x values.

Sxy

You can get the Covariance for (x,y) values.

μ

You can get the Population-Mean for x values.

σ2

You can get the Population-Variance for x values.

σ

You can get the Uncertainty for x values.

x + Δx

You can get the "μ ± σ" for x values. It means the range of uncertainty for x values.
